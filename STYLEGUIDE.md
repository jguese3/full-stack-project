# Style Guide

- We will be transitioning to tailwind for sprint 2 so most of this stuff will be updated 

## Colors Used in App

| Name | Hex | Where |
|------|-----|-------|
| **Dark Navy** | `#0f141a` | Game review background |
| **Dark Slate** | `#1e293b` | Backlog & friend status background |
| **Dark Navy 2** | `#0f172a` | Dark background gradient |
| **Purple Accent** | `#8b5cf6` | Borders, accents (backlog, avatars) |
| **Rose Red** | `#f43f5e` | Ratings |
| **Light Text** | `#e2e8f0` | Main text |
| **Bright White** | `#f1f5f9` | Emphasized text |
| **Light Gray** | `#cbd5e1` | Secondary text |
| **Medium Gray** | `#9ca3af` | Metadata, footer |
| **Dark Gray** | `#6b7280` | Dates, small text |

## Gradients

- **h1 Purple**: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- **Backlog**: `linear-gradient(135deg, #1e293b 0%, #0f172a 100%)`
- **Game Review**: `linear-gradient(135deg, #0f141a 0%, #1a1f2e 100%)`
- **Friend Status**: `linear-gradient(135deg, #1e293b 0%, #0f172a 100%)`

## Typography

| Tag | Size | Weight |
|-----|------|--------|
| **h1** | 2.5rem | 700 |
| **h2** | 2rem | 600 |
| **Large** | 1.1rem | 400 |
| **Regular** | 1rem | 400 |
| **Small** | 0.9rem | 400 |
| **Tiny** | 0.8rem | 400 |

## Spacing

| Use | Size |
|-----|------|
| **Component padding** | 2.5rem |
| **Card padding** | 1.5rem |
| **List item padding** | 1.25rem - 1.5rem |
| **Gap between items** | 1rem - 1.5rem |
| **Margins** | 2rem |

## Border Radius

- **Containers**: 16px
- **Cards**: 12px
- **Small elements**: 10px
- **Avatars**: 50%

## Shadows

- **Main**: `0 8px 24px rgba(0, 0, 0, 0.3)`
- **Hover**: `0 4px 12px rgba(139, 92, 246, 0.3)` or `0 8px 16px rgba(0, 0, 0, 0.2)`
- **Avatar**: `0 4px 8px rgba(0, 0, 0, 0.3)`

## Component Details

**Backlog Items:**
- Left border: 4px purple
- Hover: Slide right 8px

**Review Cards:**
- Background: `rgba(255, 255, 255, 0.03)`
- Border: 1px `#1f2937`
- Hover: Lift up slightly

**Friend Status Items:**
- Background: `rgba(255, 255, 255, 0.05)`
- Avatar: 64px circle, 3px purple border
- Hover: Lift up 4px
